# Project Memory

This folder is the working memory for the daily Brass Hand Oil experiment loop.

## Daily Loop

1. Write or update tests first.
2. Build one weird web experiment.
3. Record what shipped in `memory/experiments/`.
4. Append review notes to `memory/feedback-log.md`.
5. Promote durable lessons into `memory/skills/`.

## What Goes Where

- `experiments/`: one note per experiment, including concept, tests, and follow-ups
- `feedback-log.md`: a chronological log of user feedback and decisions
- `skills/`: reusable patterns, heuristics, and hard-won lessons
- `templates/`: copy-ready structures for future daily runs

## Rules

- Keep entries concise and useful.
- Store decisions in the repo, not only in chat.
- When feedback changes taste or direction, update the relevant skill note too.
