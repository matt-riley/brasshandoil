# Alias Accretion Satire

Use this pattern when a joke should become obvious through repeated interaction.

## Pattern

1. Give the visitor one clear action with a reasonable label.
2. Make each click add visible semantic actors that contradict the promised result.
3. Keep one stable metric unchanged so the failure remains legible.
4. Put detailed actors in the DOM and use canvas only for low-cost atmosphere.
5. Cap persistent actors and remove transient actors after their animation ends.

## Acceptance Checks

- Test the contradictory metric change at the interaction level.
- Add a narrow viewport overflow assertion.
- Include a reduced-motion branch.
- Keep news inspiration in notes, not in the artwork copy.

## Example

In Ghost Kitchen District, ordering food reveals two more restaurant aliases and
dispatches couriers while delivered meals stays at zero.
