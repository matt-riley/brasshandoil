# News-Seeded Experiment Concepts

- Start each daily experiment by skimming 1–3 fresh stories from a reliable source (e.g. BBC) and extract one concrete “seed constraint”:
  - **Object** (what): a device, material, animal, artifact, document
  - **Place** (where): a city, coastline, orbit, museum, court, stadium
  - **Verb** (action): evacuate, negotiate, bloom, fracture, orbit, translate
- Use the seed constraint to choose UI/interaction first, then theme second (avoid reusing the same narrative wrapper).
- Keep the citation out of the artwork; capture the inspiration link/title in the experiment memory entry instead.
- A good daily prompt: “Make the browser behave like *the thing in the headline* for 30 seconds.”
