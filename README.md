# Brass Hand Oil

Brass Hand Oil is a daily web-art playground for weird, wonderful browser experiments.

## Working Rhythm

- Build one new experiment per day.
- Cap each session at one hour.
- Use strict TDD for each experiment.
- Capture feedback, learnings, and reusable patterns so the next session starts smarter.

## Memory System

The project memory lives in [memory/README.md](/Users/mattriley/Documents/projects/personal/brasshandoil/memory/README.md). It includes:

- daily experiment notes
- a running feedback log
- reusable skills and patterns
- templates that keep future sessions consistent
